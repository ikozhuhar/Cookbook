### 1. Добавление и подготовка нового диска

```ruby
	fdisk -l
```


### 2. Разметка и форматирование диска

```ruby
	fdisk /dev/sdb
	mkfs.xfs /dev/sdb1
```


### 3. Перенос данных

```ruby
	# Временно примонтируйте новый диск
	mkdir /mnt/newdisk
	mount /dev/sdb1 /mnt/newdisk

	# Скопируйте данные
	cp -a /home/bitrix/* /mnt/newdisk/

	# Проверьте копирование
	ls -la /home/bitrix/ | wc -l
	ls -la /mnt/newdisk/ | wc -l
	du -sh /home/bitrix/
	du -sh /mnt/newdisk/
```


### 4. Изменение монтирования

```ruby
	# Переименуйте старую директорию
	mv /home/bitrix /home/bitrix.old

	# Создайте новую пустую директорию
	mkdir /home/bitrix

	# Размонтируйте временный диск
	umount /mnt/newdisk

	# Измените /etc/fstab, добавив строку:
	echo "UUID=<UUID_ВАШЕГО_ДИСКА> /home/bitrix xfs defaults 0 0" >> /etc/fstab

	# Пример (замените UUID на ваш):
	# UUID=abcd1234-5678-90ef-1234-567890abcdef /home/bitrix xfs defaults 0 0
```


### 5. Применение изменений
```ruby
	# Обновите systemd
	systemctl daemon-reload

	# Примонтируйте новый диск
	mount /home/bitrix

	# Проверьте монтирование
	df -h /home/bitrix
	ls -la /home/bitrix/

	# Если все правильно, удалите старую директорию (после проверки)
	# rm -rf /home/bitrix.old
```
