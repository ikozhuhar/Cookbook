В Linux переменные окружения хранятся и управляются несколькими способами. Вот подробное руководство:

### 1. Просмотр переменных окружения

#### _Просмотр всех переменных_

```ruby
# Все переменные (системные + пользовательские)
printenv
env

# Конкретная переменная
echo $PATH
printenv PATH

# С помощью echo
echo $HOME
echo $USER
```

#### _Просмотр в разных контекстах_

```ruby
# Только системные переменные
systemctl show-environment

# Просмотр с подробной информацией
set
```


### 2. Расположение файлов с переменными

```ruby
# Системные файлы (для всех пользователей)
/etc/environment - базовые системные переменные
/etc/profile - выполняемый при входе в систему
/etc/profile.d/ - каталог с дополнительными скриптами
/etc/bash.bashrc - для Bash (некоторые дистрибутивы)

# Пользовательские файлы
~/.bashrc - для Bash при запуске интерактивной оболочки
~/.bash_profile - для Bash при логине
~/.profile - общий профиль пользователя
~/.pam_environment - переменные через PAM
```


### 3. Временное изменение переменных

#### _На текущей сессии_

```ruby
# Установка переменной
export MY_VAR="значение"

# Добавление к PATH
export PATH=$PATH:/новая/директория

# Для одной команды
MY_VAR="значение" команда
```


### 4. Постоянное изменение переменных

#### _Способ 1: Через ~/.bashrc_

```ruby
# Редактируем файл
nano ~/.bashrc

# Добавляем в конец
export MY_VAR="моё значение"
export PATH=$PATH:/мой/путь

# Применяем изменения
source ~/.bashrc
# или
. ~/.bashrc
```


#### _Способ 2: Через ~/.profile_

```ruby
nano ~/.profile

# Добавляем export команды
export MY_VAR="моё значение"
```

#### _Способ 3: Через /etc/environment (требует root)_

```ruby
sudo nano /etc/environment

# Формат: ПЕРЕМЕННАЯ=значение (без export)
```

### 5. Удаление переменных

```ruby
# Удаление из текущей сессии
unset MY_VAR

# Удаление из файла конфигурации
# Отредактировать соответствующий файл и выполнить source
```

### 6. Практические примеры

#### _Пример настройки JAVA_HOME_

```ruby
# Временная установка
export JAVA_HOME=/usr/lib/jvm/java-11-openjdk

# Постоянная установка (в ~/.bashrc)
echo 'export JAVA_HOME=/usr/lib/jvm/java-11-openjdk' >> ~/.bashrc
source ~/.bashrc
```

#### _Пример добавления в PATH_

```ruby
# Добавление кастомного пути
export PATH=$PATH:$HOME/bin

# В файле ~/.bashrc
echo 'export PATH=$PATH:$HOME/bin' >> ~/.bashrc
```

### 7. Проверка приоритета загрузки

```ruby
Переменные загружаются в следующем порядке:

/etc/environment
/etc/profile
~/.profile или ~/.bash_profile
~/.bashrc
```

### 8. Важные системные переменные

```ruby
echo $USER        # Текущий пользователь
echo $HOME        # Домашняя директория
echo $PWD         # Текущая рабочая директория
echo $SHELL       # Текущая оболочка
echo $LANG        # Языковая настройка
echo $PS1         # Приглашение командной строки
```
